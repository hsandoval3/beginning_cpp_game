# beginning_cpp_game
Working on the "Beginning Game Programming v2.0" tutorial found here: https://github.com/tuvtran/project-based-learning.

### Tutorial Notes:
To fix SDL2 include error:  
include <SDL2/SDL.h>

To compile:  
``` shell
mkdir build && cd build
cmake ..
make
```

### Might need some software
libsdl2-dev (maybe)
libsdl2-2.0-0
cmake
clang
clang tidy
